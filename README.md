# Hidden Student Burnout Detection System

## Problem Statement
Students often experience stress, anxiety, and academic burnout which negatively affects their mental health and performance.

## Solution
This AI-based system detects student burnout levels using behavioral and emotional indicators such as sleep, stress, workload, and mood patterns.

## Features
- Burnout prediction
- Stress analysis
- Mood tracking
- Personalized suggestions
- Dashboard visualization
